# Flutter tvOS Engine Monorepo

Monorepo containing all Flutter tvOS engine components from [LibertyGlobal](https://github.com/LibertyGlobal), currently at version **3.27.4**.

## Structure

| Directory    | Repository | Description |
|-------------|-----------|-------------|
| `engine/`   | [flutter-tvos-engine](https://github.com/LibertyGlobal/flutter-tvos-engine) | Main Flutter engine for tvOS |
| `skia/`     | [flutter-tvos-skia](https://github.com/LibertyGlobal/flutter-tvos-skia) | Skia 2D graphics library |
| `perfetto/` | [flutter-tvos-perfetto](https://github.com/LibertyGlobal/flutter-tvos-perfetto) | Perfetto performance tracing |
| `buildroot/`| [flutter-tvos-buildroot](https://github.com/LibertyGlobal/flutter-tvos-buildroot) | Build environment for the engine |
